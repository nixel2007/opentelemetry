# Публичный интерфейс библиотеки `opentelemetry`

## Ядро

* [ОтелSdk](./Ядро/ОтелSdk.md) — главный класс SDK, единая точка входа в телеметрию
* [ОтелАтрибуты](./Ядро/ОтелАтрибуты.md) — хранилище пар ключ-значение для телеметрии
* [ОтелРесурс](./Ядро/ОтелРесурс.md) — описание сущности, производящей телеметрию
* [ОтелBaggage](./Ядро/ОтелBaggage.md) — передача контекстных данных между сервисами (W3C Baggage)
* [ОтелПостроительSdk](./Ядро/ОтелПостроительSdk.md) — построитель SDK (fluent API)
* [ОтелПостроительРесурса](./Ядро/ОтелПостроительРесурса.md) — построитель ресурса (fluent API)
* [ОтелПостроительBaggage](./Ядро/ОтелПостроительBaggage.md) — построитель Baggage (fluent API)
* [ОтелОбластьИнструментирования](./Ядро/ОтелОбластьИнструментирования.md) — InstrumentationScope: описание библиотеки инструментации
* [ОтелКонтекст](./Ядро/ОтелКонтекст.md) — модуль управления текущим контекстом выполнения
* [ОтелГлобальный](./Ядро/ОтелГлобальный.md) — модуль глобального реестра OpenTelemetry
* [ОтелРезультатЭкспорта](./Ядро/ОтелРезультатЭкспорта.md) — результат асинхронной операции экспорта
* [ОтелРезультатЗакрытия](./Ядро/ОтелРезультатЗакрытия.md) — результат операции закрытия / принудительной выгрузки
* [ОтелКлючКонтекста](./Ядро/ОтелКлючКонтекста.md) — ключ для хранения значений в контексте
* [ОтелТокенКонтекста](./Ядро/ОтелТокенКонтекста.md) — токен, возвращаемый при присоединении контекста

## Трассировка

* [ОтелПровайдерТрассировки](./Трассировка/ОтелПровайдерТрассировки.md) — провайдер трассировки SDK
* [ОтелПостроительПровайдераТрассировки](./Трассировка/ОтелПостроительПровайдераТрассировки.md) — построитель провайдера трассировки
* [ОтелТрассировщик](./Трассировка/ОтелТрассировщик.md) — создание и управление спанами
* [ОтелСпан](./Трассировка/ОтелСпан.md) — единица работы или операции в трассировке
* [ОтелПостроительСпана](./Трассировка/ОтелПостроительСпана.md) — построитель спана (SpanBuilder)
* [ОтелКонтекстСпана](./Трассировка/ОтелКонтекстСпана.md) — контекст спана (SpanContext)
* [ОтелСобытиеСпана](./Трассировка/ОтелСобытиеСпана.md) — событие в рамках спана (SpanEvent)
* [ОтелЛимитыСпана](./Трассировка/ОтелЛимитыСпана.md) — лимиты количества атрибутов, событий и линков
* [ОтелЛинк](./Трассировка/ОтелЛинк.md) — связь спана с другим спаном
* [ОтелСостояниеТрассировки](./Трассировка/ОтелСостояниеТрассировки.md) — W3C TraceState
* [ОтелВидСпана](./Трассировка/ОтелВидСпана.md) — модуль констант видов спана (SpanKind)
* [ОтелКодСтатуса](./Трассировка/ОтелКодСтатуса.md) — модуль констант кодов статуса спана
* [ОтелСэмплер](./Трассировка/ОтелСэмплер.md) — модуль стратегий семплирования
* [ОтелПростойПроцессорСпанов](./Трассировка/ОтелПростойПроцессорСпанов.md) — синхронный процессор спанов
* [ОтелПакетныйПроцессорСпанов](./Трассировка/ОтелПакетныйПроцессорСпанов.md) — пакетный асинхронный процессор спанов

## Логирование

* `ОтелПровайдерЛогирования` — провайдер логирования SDK
* `ОтелЛоггер` — логгер для записи событий
* `ОтелЗаписьЛога` — запись журнала (LogRecord)
* `ОтелПростойПроцессорЛогов` — синхронный процессор записей журнала
* `ОтелПакетныйПроцессорЛогов` — пакетный процессор записей журнала
* `ОтелПостроительЛоггера` — построитель логгера
* `ОтелПостроительПровайдераЛогирования` — построитель провайдера логирования
* `ОтелСтепеньСерьезности` — константы степени серьезности лог-записи

## Метрики

* `ОтелПровайдерМетрик` — провайдер метрик SDK
* `ОтелМетр` — метр для создания инструментов метрик
* `ОтелСчетчик` — монотонно возрастающий счетчик (Counter)
* `ОтелРеверсивныйСчетчик` — двунаправленный счетчик (UpDownCounter)
* `ОтелДатчик` — синхронный датчик значений (Gauge)
* `ОтелГистограмма` — гистограмма распределения (Histogram)
* `ОтелПостроительМетра` — построитель метра
* `ОтелПостроительПровайдераМетрик` — построитель провайдера метрик
* `ОтелПериодическийЧитательМетрик` — периодическое чтение метрик для экспорта

## Экспорт

* `ОтелЭкспортерСпанов` — OTLP-экспортер спанов
* `ОтелЭкспортерЛогов` — OTLP-экспортер записей журнала
* `ОтелЭкспортерМетрик` — OTLP-экспортер метрик
* `ОтелHttpТранспорт` — HTTP/JSON транспорт для OTLP
* `ОтелGrpcТранспорт` — gRPC транспорт для OTLP
* `ОтелВПамятьТранспорт` — транспорт в памяти (для тестирования)
* `ОтелНастройкиTls` — настройки TLS-соединения

## Пропагация

* `ОтелW3CПропагатор` — пропагатор W3C Trace Context
* `ОтелW3CBaggageПропагатор` — пропагатор W3C Baggage
* `ОтелКомпозитныйПропагатор` — составной пропагатор
* `ОтелНоопПропагатор` — пустой пропагатор (noop)
* `ОтелГеттерТекстовойКарты` — вспомогательный класс для чтения заголовков
* `ОтелСеттерТекстовойКарты` — вспомогательный класс для записи заголовков

## Конфигурация

* [ОтелАвтоконфигурация](./Конфигурация/ОтелАвтоконфигурация.md) — автоматическая инициализация SDK из переменных окружения
